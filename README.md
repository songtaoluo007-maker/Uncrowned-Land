# Uncrowned Land / 无冠之地

一款以个人崛起、伙伴培养、轻量城建、佣兵雇佣、职业分工和赛季战争为核心的多人策略 RPG 手游。

> 有人建立王国，有人为王国征战，还有人只为出得起价的人而战。

## 当前阶段

项目处于立项与首个垂直切片阶段，暂不开发完整国家战争和大规模联机。

首个验证闭环：

1. 从无名者开始；
2. 招募并命名普通伙伴；
3. 组建小队并完成战斗；
4. 获得村庄；
5. 任命伙伴管理村庄；
6. 在领主与佣兵两条未来路线之间做出选择。
